# SecureBank AI - Hybrid Banking Chatbot

## 📌 Project Overview
SecureBank AI is a smart banking chatbot that provides accurate responses using a hybrid approach combining structured JSON data and AI.

## 🚀 Features
- User Login System
- Banking Chatbot Interface
- JSON Knowledge Base (Loans, Credit Cards, EMI)
- AI Integration using Ollama (Llama3)
- Hybrid Response System (JSON + AI)
- Conversation History

## 🛠 Technologies Used
Python, Streamlit, Ollama, Llama3, JSON

## ⚙️ How It Works
1. User asks a query
2. System checks JSON data
3. If found → returns exact answer
4. Else → sends to AI (Ollama)
5. Displays response

## ▶️ How to Run
1. Install dependencies:
   pip install streamlit ollama

2. Run the app:
   streamlit run Fin.py

3. Make sure Ollama is running:
   ollama run llama3
